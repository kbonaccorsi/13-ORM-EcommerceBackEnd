# 13 Object-Relational Mapping (ORM): E-Commerce Back End

GitHub repo: https://github.com/kbonaccorsi/13-ORM-EcommerceBackEnd

walkthrough videos:
  1: https://watch.screencastify.com/v/BcCjzT4cTtSRi2mRHN21
  2: https://watch.screencastify.com/v/dC78TrCLmehN1rkZdFcH


## DESCRIPTION

Build the back end for an e-commerce site by modifying starter code. Be able to get/post/put/delete to categories, products, and tags.  Be able to get/put/delete by using id numbers.


## INSTALL

You will need to npm install  to get the necessary packages to make the app function as expected.  Then, npm run seed to populate the information, followed by npm run start to enable running fetch calls.
